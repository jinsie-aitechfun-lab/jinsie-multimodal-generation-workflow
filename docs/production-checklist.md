# AI Video Studio 生产环境验收清单

## 目的

用于每次部署到生产环境后的快速验收。

覆盖： - Workflow 创建 - 图片生成 - 图片审核 - 图片重新生成 - Selector
选择 - 状态恢复 - 视频生成 - 字幕 - 配音 - 历史作品 - 移动端体验 -
前端加载体验

------------------------------------------------------------------------

# 验收项目

## 1. 基础环境检查

-   [ ] 前端页面可正常访问
-   [ ] 后端 API 健康检查正常
-   [ ] 静态资源可访问
-   [ ] 视频资源可访问
-   [ ] 图片资源可访问

## 2. Landing → Studio 进入体验

-   [ ] 点击"开始创作"后有进入反馈
-   [ ] 不出现长时间白屏
-   [ ] Studio 页面正常加载
-   [ ] 首次进入和再次进入体验一致
-   [ ] 移动端首次进入无横向滚动
-   [ ] 布局不会首次加载异常变宽

## 3. Workflow 创建

-   [ ] workflow 创建成功
-   [ ] workflowId 正常保存
-   [ ] runId 正常保存
-   [ ] 刷新页面后仍能恢复当前任务

## 4. 图片生成

-   [ ] 多场景正常生成
-   [ ] 图片逐步出现
-   [ ] 单个场景完成后状态更新
-   [ ] readyCount 正常增加
-   [ ] 不出现长时间 0% 无反馈

## 5. 图片恢复

-   [ ] 离开 Studio 后重新进入可继续生成
-   [ ] 不重新创建图片任务
-   [ ] 不重复调用 provider

## 6. 图片重新生成

-   [ ] 点击重新生成会创建新任务
-   [ ] 新候选图片不同于旧候选
-   [ ] 不复用旧 succeeded task

## 7. Selector

-   [ ] selector 评分结果被保留
-   [ ] 高评分候选可以正确选择
-   [ ] 不固定选择第一张图片
-   [ ] refresh 后不丢失选择结果

## 8. 图片缩略图

-   [ ] 优先加载 thumbnail
-   [ ] 不直接加载大 PNG
-   [ ] 加载失败有 fallback
-   [ ] 历史图片不会大量破图

## 9. 视频生成

-   [ ] 图片完成后进入视频生成
-   [ ] 配音生成正常
-   [ ] 字幕生成正常
-   [ ] 视频合成成功
-   [ ] 播放器 metadata 正常加载

## 10. 视频恢复

-   [ ] 离开页面后不会重复 render
-   [ ] 返回后可恢复视频状态
-   [ ] 视频完成后自动显示

## 11. 字幕验收

### 安全区域

-   [ ] 不超出左右边界
-   [ ] 不裁切
-   [ ] 中文正常显示
-   [ ] 英文和数字混排正常

### 同步

-   [ ] 字幕与配音一致
-   [ ] 长句无提前或滞后
-   [ ] 多段字幕切换自然

测试： - 长中文 - 无标点中文 - 中英数字混合

## 12. 历史作品

-   [ ] 视频封面正常显示
-   [ ] 重新进入后封面仍存在
-   [ ] 历史视频可播放
-   [ ] 不触发重新生成

## 13. UI 回归

-   [ ] 灵感参考弹窗有遮罩
-   [ ] 深色主题正常
-   [ ] 浅色主题正常
-   [ ] 页面无异常布局

## 14. 移动端专项

-   [ ] 无横向滚动
-   [ ] 底部导航正常
-   [ ] 弹窗正常
-   [ ] 图片卡片正常
-   [ ] 视频播放正常

## 15. 发布前 Git 检查

执行：

``` bash
git status
git log -1
git diff --check
```

确认：

-   [ ] 工作区干净
-   [ ] PR 已合并
-   [ ] 部署版本正确
